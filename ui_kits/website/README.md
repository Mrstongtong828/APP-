# CICADA · Website UI kit

Marketing website recreation for **CICADA 思科达 / 佛山市思科达医疗器械**. The official site is not part of the supplied codebase — this UI kit is a **brand-driven recreation** built from the same tokens, palette, type system and content patterns as the mini-program. Treat copy as illustrative; structure is what's portable.

## Files
- `index.html` — one-pager: hero, stats, services, process, brands, testimonials, contact, footer
- All components inline in the HTML for portability — sections clearly commented

## Notes / flags
- No real homepage screenshots were provided; layout is composed from brand collateral + mini-program tokens.
- Photos use the two brand JPGs (`photo-building`, `photo-factory`) with the steel-blue overlay treatment.
- Typography raised one step from the mini-program baseline (web body 17 px) since desktop reading distance is greater.
